**Data Science Projects Repository**

**Overview**

This repository is for end-to-end Data Science and ML projects combining NumPy, Pandas, Matplotlib, Scikit-learn, and more. It includes Jupyter Notebooks for data analysis, visualization, and modeling.

**Structure**

/notebooks/: Jupyter Notebooks for complete projects (e.g., titanic_full.ipynb).

/scripts/: Python scripts for reusable code.

/data/: Datasets (not tracked; store locally or link to sources).

/images/: Visualizations and model outputs.

**Setup Instructions**

    Clone the Repository:git clone https://github.com/anikashraful/data-science-projects.git


    Install Dependencies:pip install -r requirements.txt


Run Notebooks:Open .ipynb files in /notebooks/ using Jupyter Notebook or VS Code.


**Requirements**

Python 3.8+

Libraries: pandas, numpy, matplotlib, scikit-learn, jupyter,google colab

    Install via:pip install pandas numpy matplotlib scikit-learn jupyter
